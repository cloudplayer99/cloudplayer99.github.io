¶使用 VMware 安装 Ubuntu（保姆级教学）

¶先作一点小准备

¶Download the latest LTS version of Ubuntu

官方下载地址：https://ubuntu.com/download/desktop
Ubuntu 20.04.1 LTS
Download the latest LTS version of Ubuntu, for desktop PCs and laptops. LTS stands for long-term support — which means five years, until April 2025, of free security and maintenance updates, guaranteed.

¶Download VMware Workstation Pro

借助 VMware Workstation Pro，您可以在同一台 Windows 或 Linux PC 上同时运行多个操作系统。创建真实的 Linux 和 Windows 虚拟机以及其他桌面、服务器和平板电脑环境（包括可配置的虚拟网络连接和网络条件模拟），用于代码开发、解决方案构建、应用测试、产品演示等。

官方下载地址：https://www.vmware.com/cn/products/workstation-pro/workstation-pro-evaluation.html

¶开始安装 Ubuntu

可以直接去看 VMware Workstation Pro 产品文档 ，下面给出我的安装流程，仅作为参考

¶1. 打开 VMware Workstation Pro 选择创建新的虚拟机

¶2. 选择自定义（高级）类型的配置

其实选典型（推荐）类型的配置好像更方便

¶3. 硬件兼容性默认

¶4. 安装来源选择稍后安装操作系统

¶5. 客户机操作系统：Linux / 版本：Ubuntu 64 位（如果电脑是 32 位的请选择 Ubuntu）

¶6. 确定虚拟机的名称与位置

¶7. 指定处理器数量

这里的设定与性能有关，视情况而定，可以直接按默认的来，也可以适当增加，反正之后是可以再去修改的

¶8. 虚拟机的内存：2048 MB 即推荐内存 2GB

¶9. 网络类型选择使用网络地址转换（NAT）

¶10. I/O 控制器类型：默认 / 虚拟磁盘类型：默认

¶11. 选择磁盘：创建新虚拟磁盘

¶12. 最大磁盘大小：自定义 / 将虚拟磁盘存储为单个文件

选项里面作了一些说明，按自己的需求来就好

¶13. 确定磁盘文件的地址

¶14. 自定义硬件

在设备栏中点击 新 CD/DVD(SATA)
在连接一栏中选择 使用 ISO 映像文件
点击浏览，找到之前在准备过程中下载的 Ubuntu 20.04.1 LTS
文件名：ubuntu-20.04.1-desktop-amd64.iso

可以删掉（移除）打印机

设置好后点击 关闭
回到 新建虚拟机向导 ，点击 完成

此时回到了 VMware Workstation 界面
发现其已为我们创建好了一个虚拟机
点击 开启此虚拟机 ，然后开始安装 Ubuntu 前的配置

一路向前，有继续的点继续

放在这儿，让它慢慢整（指直接睡到四点）

有号上号

设置 Livepatch 的同时顺便创了一个 Ubuntu One 的号

基本完事，但是会发现全屏效果感人，于是继续

¶安装 VMware Tools

可以直接点击其下方提示中的 安装 Tools
也可以点击位于界面上方的 虚拟机(M) -> 安装 VMware Tools(T)…
下载完成后，找到图片中对应的文件并打开，将其中文件提取至桌面
在桌面上打开提取过来的文件，并进入终端
输入命令：sudo ./vmware-install.pl 并输入密码（输入时不显示）
然后在需要输入 yes 或 no 的时候总输入 yes
[yes] 是默认 yes，可以直接回车
其余要输入的都是选默认，也可以直接回车

安装完成，在上方点击 进入全屏模式 即可占满你的整个桌面
