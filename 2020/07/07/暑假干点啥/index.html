暑假干点啥

发表于 2020-07-07 更新于 2021-02-21
本文字数： 671 阅读时长 ≈ 1 分钟

¶这假放的，有点迷

放假一个星期了，除了练车，剩下的时间都嗨的一批，Steam 上又多了几个能玩的。。。

按现在 B 市的行情，9 月份正常开学怕是有点困难，宿舍里不管是起灰还是发霉，或者是长毛，都已经无所谓了，难以接受的是我扔在沙河的设备，幸好里面除了 RED DEAD REDEMPTION 2 也没啥大作，否则吃灰了半年我真的要哭死qwq

¶暑假干点啥

Jetson nano 小车也要到了，再加上搭载 STM32F103 的那台，我也是即将拥有两台车的人了/狗头

这个暑假事儿还是挺多的，博客里面两个系列还没有整理完，Verilog 那个考完了，老师一反常态的给了很高的平时分占比，也是不容易呀，后续我会把 Verilog HDL 的系列整理好，留下一点学过数电的痕迹。。。北理 ACM 冬训的专题也得弄下去，顺便把他给的练习题的题解也做一下；模电的仿真也好久没做了，想法是把书上的仿真题全部过一遍，也做一点知识点上的整理；最近还做了一个C++的集合、二元关系、图的类的实现，做得差强人意，里面还存在一点 bug ，稍微整理一下又可以水一篇博客/大笑

离散数学、数据结构、算法导论，这仨就够我喝一壶的了，平时再刷刷洛谷、经典里的题，打好基础吧

复变与电路下还等着回学校考呢，还得复习一波qwq

Python 也得重新捡起来，Jetson nana 就得用了，听说下个学期还得学，嗯，就是教数电的老师教 Python

在 CSU 学软工的大佬把他们好久以前的做雷霆战机打了个包给我，就作为我学 C++ 的第一个项目吧

对了，信号与系统，下个学期的大 Boss，是不是给点面子预习一下？

Flag 立下来，这两个月必弄不完
