下几个软件搞搞正事

发表于 2023-11-04 更新于 2023-11-07
本文字数： 4.1k 阅读时长 ≈ 4 分钟

在我的新电脑（半年了没干什么正事）上下点要用的软件

¶LaTeX

先下一个 LaTeX
即下一个 TeX 排版引擎，TeX 的发行版有 Tex Live 和 MiKTeX
这里下 TeX Live，还要为 LaTeX 选一款编辑器，这里决定把 VSCode 的环境配置一下，并且下一个 TeXstudio 集成环境【这是 TeXstudio 的介绍】

一个非常快速的 Latex 入门教程
这个视频中提到了 TeX Live 的下载地址，LaTeX 公式编辑器 与一份【一份（不太）简短的 LaTeX 介绍】，这份文档打印出来很久了，一直没怎么看，现在用上了

打开这个网站 TeX Live 的下载地址
点 installing TeX Live over the Internet 把安装包 install-tl-windows.exe 下下来

安装时间较长。。。

下载一个名为 LaTeX Workshop 的插件

接着是下 TeXstudio
进 TeXstudio 的官网，直接点下载
这里下的是 texstudio-4.6.3-win-qt6.exe，直接安装
这里下完就直接是默认的中文了，不用按教程再改成中文
注：在 Options > Configure TeXstudio 的 General 里面设置 Language 为 zh_CN (Chinese)

OK，LaTeX 的部分完了，之后再看看文献管理工具 Zotero 是个什么情况

¶CAJViewer

下一个 CAJ 阅读器，看知网论文用
访问 cajviewer 浏览器官网，直接下载

¶知云

下一个知云，翻译英文文献用
访问 知云文献翻译官网，直接下载
两个翻译引擎也够用

¶Visio

Visio 用来画图，论文、博文、笔记啥的图都可以，配合 PPT 使用，两个在画图上都不错
在微信公众号软件管家里弄到百度网盘的压缩包
office 和 visio 要共存，需要相同位数的安装包，这里我的 office 都是 64 bit 的
所以 visio 也下 64 bit 的
按照公众号的教程来，visio 21 就装好了，注意电脑里的 windows defender

¶Matlab

Matlab 最近的机械优化设计课上又要用了，赶紧下一下
还是在微信公众号软件管家里弄到百度网盘的压缩包，之后的软件都在这里面弄了
（因为都下不了正版，只能下盗版、破解版）
版本为 MatlabR2023b
重要的是它给的安装密钥与许可证文件 licese.lic，最后再找到安装路径将 win64 文件进行替换

¶Altium Designer

AD23 用来画板子，目前还啥都不会

¶Multisim、 Proteus 和 Keil

这哥仨还不急

¶SolidWorks

这个比较重量级，机械学生总要会一样机械的工程软件吧
安装起来比较麻烦，一步步来就好

¶Zotero

Zotero 官网
Zotero 6 for Windows 与 Zotero Connector 这两个东西都要下下来
其中 Zotero Connector 是 Chrome 浏览器的插件，方便在 Chrome 上浏览论文的时候加入自己的文献库
为了在写 LaTeX 的时候结合 Zotero，需要下一个叫 Better-BibTeX 的插件，并安装在 Zotero 上
下载地址在 github 的 zotero-better-bibtex 开源仓库
下它最新的发行版就行，这里下载的是 zotero-better-bibtex-6.7.136.xpi
为了能够配合 VSCode 使用，在 VSCode 中安装扩展 Zotero LaTeX
安装后将设置中的 Zotero: LaTeX Command 一项改为 cite
